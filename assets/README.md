# Portfolio — Assets Folder

Place the following files here:

| File           | Description                                         |
|----------------|-----------------------------------------------------|
| `cv.pdf`       | Your CV in PDF format. Linked from all download buttons. |
| `profile.jpg`  | Your profile photo. Shown in the About section. If missing, a placeholder icon is shown automatically. |

## How to add your CV

1. Export your CV as a PDF.
2. Name the file `cv.pdf`.
3. Drop it into this `assets/` folder.
4. The download buttons throughout the page will automatically serve it.
